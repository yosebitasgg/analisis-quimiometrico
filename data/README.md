# Carpeta de Datos

Esta carpeta debe contener el archivo de dataset de ejemplo:

```
chemometrics_example.xls
```

## Nota importante

El archivo `chemometrics_example.xls` debe estar ubicado en esta carpeta para que funcione el botón "Usar dataset de ejemplo" en la aplicación.

Si el archivo no está presente, puedes:

1. Colocar el archivo Excel con los datos de FAMEs aquí
2. O subir manualmente cualquier archivo CSV/Excel desde la interfaz

## Formato esperado

El dataset debe tener:
- Columnas numéricas con las áreas de picos (variables para el análisis)
- Opcionalmente, columnas `feedstock` y `concentration` como variables categóricas
